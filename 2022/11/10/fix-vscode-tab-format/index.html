修改VSCODE的TAB为空格

Visual Studio Code 是非常好用的代码编辑器，包括常用的编程语言。但是其设置相对其他IDE稍微复杂了一些，今天就遇到了格式化后代码使用了TAB缩进，而不是我之前设置的空格模式，毕竟空格模式在其他IDE下最终显示效果是不变的， 有些IDE的TAB设置的是8个空格哦。

修改VSCODE的TAB为空格

其实修改VSCODE的TAB为空格，非常简单，如果你在设置里面去找，可能还不太容易找到，反正我是没找到。
后来我发现，在IDE的右下角就可以直接更改，非常方便。
首先在右下角选择类似 制表符长度:4 的提示，单击就行，如图

点击之后，标题栏下方会出现提示框，我们选择第一个，如图

这个时候出现了你需要配置的空格大小，默认是IDE配置的TAB长度，比如4，点击就可以了。
